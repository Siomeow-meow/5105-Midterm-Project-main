# MFA Authentication Website

A secure web application demonstrating Multi-Factor Authentication using Google Authenticator/Microsoft Authenticator.

## Features

- User registration and login
- MFA setup with QR codes
- TOTP (Time-based One-Time Password) verification
- Secure session management
- Responsive design
